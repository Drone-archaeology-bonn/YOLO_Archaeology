# Dataloader

## Code

### Extracting_parameters.ipynb

Takes a GeoJSON and returns all parameters, that are important for labeling the features, that YOLOv5 needs, as a list.

Input: GeoJSON

Output: List [class, x_center, y_center, width, height]
